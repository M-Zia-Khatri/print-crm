# CRM Modules

Implemented foundation modules:

- Authentication and profile settings from the starter kit
- Dashboard metrics
- Users, roles, and permissions schema
- Customers with list/create flow
- Companies
- Products and services
- Quotes
- Invoices
- Payments
- Tasks
- Notes
- Activity logs
- Settings

Non-customer modules currently have production route/page placeholders so each can be expanded without changing navigation or route conventions.
